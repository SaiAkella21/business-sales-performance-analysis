
# Business Sales Performance Analytics Dashboard

## Overview
This project analyzes business sales performance using Power BI. The dashboard provides insights into sales trends, holiday sales impact, store performance, and unemployment analysis.

## Tools Used
- Power BI
- Excel Dataset

## Dashboard Features
- KPI Cards
- Weekly Sales Trend Analysis
- Holiday vs Non-Holiday Sales Comparison
- Store Performance Comparison
- Unemployment Scatter Analysis

## Key Insights
- Holiday periods generated higher sales.
- Some stores consistently outperformed others.
- Sales trends fluctuated over time.
- Unemployment showed moderate influence on weekly sales.
